# LexYacc in Ubuntu
  
  Welcome to the LexYacc!

***
Setup procedure Linux os:

* install lex and bison
* open terminal and run code like....
***
Run lex code:
* lex four.l
* gcc -o four lex.yy.c
* ./four
***
Run lex and yacc:
* lex one.l
* yacc -d one.y
* cc y.tab.c lex.yy.c -ll
* ./a.out
